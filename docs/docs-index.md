**棚卸しルール**  
- 月次またはリリース前に `rg --files -g '*.md'` を実行し、索引と実ファイルに差異がないか確認する。  
- 新しい検証ログを `docs/test-notes/` に追加した場合は、本索引と `docs/AGENTS.md` のリンク集を更新する。
